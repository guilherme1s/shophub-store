# ShopHub

ShopHub é uma aplicação de e-commerce moderna e responsiva, desenvolvida para se aproximar de uma loja online real. O projeto foca em interface bem construída, código organizado e um fluxo de compra completo, inspirado em plataformas como Shopee, AliExpress e Mercado Livre.

Na aplicação, é possível navegar por diferentes categorias, visualizar detalhes dos produtos, adicioná-los ao carrinho e simular todo o processo de compra de forma clara e funcional.
